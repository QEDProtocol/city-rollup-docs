# Register User

## Description
Register a new user on the rollup chain. 
Users should provide an L2 private key for registration. If not, a random wallet can be generated through the `random-wallet` command.

## Usage
```bash
city-rollup-user-cli register-user --private-key <L2 private key>
```

## Example
Register a new user with a private key 0ed2be709b9ac1cefb7c35449c423efd22bbfe5cff7aa08f00faad772c242a34

```bash
city-rollup-user-cli register-user --private-key 0ed2be709b9ac1cefb7c35449c423efd22bbfe5cff7aa08f00faad772c242a34         
```

## Note
- The `rpc_address` can use the environment variable `RPC_ADDRESS` to set the RPC address.
