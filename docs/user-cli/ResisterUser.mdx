# Register User

## Description
Register a new user on the rollup chain. User should provide a private key to register on the rollup chain.
## Usage
```bash
city-rollup-user-cli register-user --rpc_address <rpc_address> --private_key <private_key>
```

## Example
Register a new user with a private key 0ed2be709b9ac1cefb7c35449c423efd22bbfe5cff7aa08f00faad772c242a34

```bash
city-rollup-user-cli register-user --rpc_address http://127.0.0.1:8545 --private-key 0ed2be709b9ac1cefb7c35449c423efd22bbfe5cff7aa08f00faad772c242a34
```

## Note
- The `rpc_address` can use the environment variable `RPC_ADDRESS` to set the RPC address.
