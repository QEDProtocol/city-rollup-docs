# Get Public Key

## Description
Get the public key from the private key.

## Usage
```bash
city-rollup-user-cli get-public-key --private_key <private_key>
```

## Example
Get the public key from the private key 0ed2be709b9ac1cefb7c35449c423efd22bbfe5cff7aa08f00faad772c242a34
```bash
city-rollup-user-cli get-public-key --private-key 0ed2be709b9ac1cefb7c35449c423efd22bbfe5cff7aa08f00faad772c242a34
```
