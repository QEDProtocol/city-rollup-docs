# Generate Random Wallet

## Description
Generate a random wallet with no private key or seed. The private key is generated randomly and is not stored in the wallet. The private key is displayed in the console and should be stored securely by the user. The wallet address is displayed in the console and can be used to interact with the rollup chain.
## Usage
```bash
city-rollup-user-cli random-wallet --private_key <private_key>
```

## Example
Generate a wallet with a private key 0ed2be709b9ac1cefb7c35449c423efd22bbfe5cff7aa08f00faad772c242a34
```bash
city-rollup-user-cli random-wallet --private_key 0ed2be709b9ac1cefb7c35449c423efd22bbfe5cff7aa08f00faad772c242a34
```
